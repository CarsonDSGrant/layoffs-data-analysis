# Layoffs Data Analysis Project

## Project Overview
End-to-end SQL data cleaning and analysis of global layoffs data from Kaggle. This project demonstrates comprehensive data wrangling, exploratory analysis, and business intelligence reporting.

## Technologies Used
- MySQL / PostgreSQL
- Advanced SQL (CTEs, Window Functions, Self-Joins)
- Data Cleaning & Validation Techniques

## Project Structure
- **`data_cleaning.sql`** - Complete pipeline from raw data to analysis-ready dataset
- **`data_analysis.sql`** - Exploratory analysis and business insights
- **`sample_data/`** - Sample dataset for demonstration

##  Key Findings
- Identified companies with 100% workforce reduction and analyzed patterns
- Calculated rolling layoff totals showing temporal trends
- Ranked industries and countries by layoff severity
- Discovered relationships between funding raised and layoff percentages

## How to Use
1. Download the dataset at layoffs.csv
2. Run `data_cleaning.sql` to prepare the data
3. Execute `data_analysis.sql` for insights

## Skills Demonstrated
- Data cleaning & standardization
- Complex SQL queries (ROW_NUMBER, PARTITION BY, CTEs)
- Time-series analysis
